# Auto-Committer

This repo helps a lazy dev to push updates in your Github account without coding heavily. Just run the script and you won't miss any green boxes in your Github status.

## How to Use

1. Create your own repo in your Github account. **[SUGGESTED: Auto-Commiter]** <br>
2. Modify the Git credentials based on your account. Then, run init.sh in your terminal to connect yout local to remote repo. <br>
3. You can now run the auto-commit.sh! Change the *_NUM_OF_ITERATIONS_* for the desired number of loops in commits.
